Notice:
   Before use the DAQNavi Device ActiveX Control examples for Visual Basic 6, please:
   1. Install Visual Basic 6.0 SP6;
   2. Register the DAQNavi Device ActiveX Control, which is automatically done by DAQNavi SDK installer;
   3. Copy "SimpleGraphDLL.dll" and "AniGIF.ocx" to the "C:\WINDOWS\system32" directory and register "AniGIF.ocx".
   4. For Win8.1 and version higher than 8.1, it has no "mscomctl.ocx", please copy this file to "C:\WINDOWS\system32" directory and register.
